# Calculadora Cliente-Servidor con Sockets

## Descripción
Este proyecto implementa una aplicación **cliente-servidor** utilizando **sockets** en Java.  
El cliente envía una operación aritmética en formato texto (por ejemplo `5+3`), y el servidor procesa la operación y devuelve el resultado.  
La calculadora soporta **suma**, **resta** y **multiplicación** entre números comprendidos entre **0 y 9**.

---

## Funcionalidades
- Conexión entre cliente y servidor mediante sockets TCP.
- Operaciones matemáticas básicas:
  - Suma (`a + b`)
  - Resta (`a - b`)
  - Multiplicación (`a * b`)
- Validación de entrada.
- Respuesta automática desde el servidor.

---

## Tecnologías utilizadas
- Java (JDK 8+)
- `ServerSocket` y `Socket`
- `BufferedReader` y `PrintWriter`
